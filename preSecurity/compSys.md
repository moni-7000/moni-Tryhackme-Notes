# Inside a Computer System

### Almost every computer has:
- Motherboard (skeleton & nerves)
- RAM (short-term memory)
- PSU (heart & lungs)
- CPU (brains)
- GPU (visual cortex)
- HDD + SSD (long term memory


#### The Motherboard
- Connects other parts 

#### RAM (Random Access Memory)
- Holds the data that the CPU needs quick access to
- Power lost -> Contents gone

#### PSU (Power Supply)
- Supplies energy to all system components
- Takes power via outlet and distributes via connectors
- If energy demand > PSU -> **FAIL**

#### CPU (Central Processing Unit)
- Does calculations and executes instructions
- The brain of the operation

#### GPU (Graphics card)
- Receives info and creates the visuals

#### HHD / SSD
- Saves data permanently even when turned off
- HHD -> Older tech, moving parts
- SSD -> Much faster 

#### Network Adapter
- Allows a computer to communicate with other computers and networks 

#### Input / Output 
- Send & receive data from ur computer
